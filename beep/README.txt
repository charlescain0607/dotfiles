A bigger bitmap font inspired by creep, with glyphs to make LaTeX formatting nicer in vim. Created with gbdfed.

Installation

put the bdf file in your .fonts folder, and run

> fc-cache -fv

to update the fonts cache. To check that it properly installed, run

> fc-list | grep beep
